# CSGOPolygon-Autobet

## Użycie
1. Prawy przycisk myszy.
2. Kliknij na "Zbadaj element"
3. Z paska u góry wybierz "Konsola"
4. Wklej to:
```
fetch("https://raw.githubusercontent.com/BlacKlyExactly/CSGOPolygon-Autobet/main/autoi-bet.js")
.then(( res ) => res.text())
.then((t) => eval(t));
```
